title Prise d'une leçon de conduite - AEL

actor Eleve
actor Secrétaire
participant Lecons.jsx
participant leconService.js
participant CalendrierController.cs
participant LeconController.cs
participant LeconMetier.cs
participant LeconRepository.cs
database BDD

Eleve -> Secrétaire: Demande un créneau

Secrétaire -> Lecons.jsx: Ouvrir la page de leçons

note over Lecons.jsx,BDD: Chargement initial : créneaux, élèves, moniteurs, modèles de véhicules
Lecons.jsx -> leconService.js: getCalendrier()
leconService.js -> CalendrierController.cs: GET /api/Calendrier
CalendrierController.cs -> BDD: SELECT DateHeure FROM Calendrier
BDD --> CalendrierController.cs: creneauxList
CalendrierController.cs --> leconService.js: 200 OK creneauxList
leconService.js --> Lecons.jsx: creneauxList

Lecons.jsx --> Secrétaire: Afficher le calendrier avec les jours disponibles

Secrétaire -> Lecons.jsx: Sélectionner un jour disponible dans le calendrier
Lecons.jsx --> Secrétaire: Afficher les créneaux horaires du jour

Secrétaire -> Lecons.jsx: Sélectionner un créneau horaire
Lecons.jsx --> Secrétaire: Afficher le formulaire de réservation

Secrétaire -> Lecons.jsx: Saisir élève, moniteur, modèle de véhicule, durée

Lecons.jsx -> leconService.js: reserver(lecon)
leconService.js -> LeconController.cs: POST /api/Lecon

LeconController.cs -> LeconMetier.cs: ReserverLecon(lecon)

LeconMetier.cs -> LeconRepository.cs: CreneauExiste(dateHLecon)
LeconRepository.cs -> BDD: SELECT COUNT(*) FROM Calendrier WHERE DateHeure = @DateH
BDD --> LeconRepository.cs: count
LeconRepository.cs --> LeconMetier.cs: bool

LeconMetier.cs -> LeconRepository.cs: MoniteurEstActif(moniteurId)
LeconRepository.cs -> BDD: SELECT COUNT(*) FROM Moniteur WHERE IdMoniteur = @Id AND Activite = 1
BDD --> LeconRepository.cs: count
LeconRepository.cs --> LeconMetier.cs: bool

LeconMetier.cs -> LeconRepository.cs: MoniteurEstDisponible(moniteurId, dateHLecon)
LeconRepository.cs -> BDD: SELECT COUNT(*) FROM Lecon WHERE MoniteurId = @Id AND DateHLecon = @DateH
BDD --> LeconRepository.cs: count
LeconRepository.cs --> LeconMetier.cs: bool

LeconMetier.cs -> LeconRepository.cs: EleveEstDisponible(eleveId, dateHLecon)
LeconRepository.cs -> BDD: SELECT COUNT(*) FROM Lecon WHERE EleveId = @Id AND DateHLecon = @DateH
BDD --> LeconRepository.cs: count
LeconRepository.cs --> LeconMetier.cs: bool

LeconMetier.cs -> LeconRepository.cs: VehiculeEstDisponible(modeleLecon, dateHLecon)
LeconRepository.cs -> BDD: SELECT COUNT(*) FROM Vehicule WHERE ModeleVehicule = @Modele AND Etat = 1
BDD --> LeconRepository.cs: nbVehicules
LeconRepository.cs -> BDD: SELECT COUNT(*) FROM Lecon WHERE ModeleLecon = @Modele AND DateHLecon = @DateH
BDD --> LeconRepository.cs: nbLecons
LeconRepository.cs --> LeconMetier.cs: nbLecons < nbVehicules

alt [Une vérification échoue]
LeconMetier.cs --> LeconController.cs: return ReservationResult.Fail(message)
LeconController.cs --> leconService.js: 409 Conflict { message }
leconService.js --> Lecons.jsx: erreur
Lecons.jsx --> Secrétaire: Afficher le message d'erreur
else [Toutes les vérifications sont OK]
LeconMetier.cs -> LeconRepository.cs: CreateLecon(lecon)
LeconRepository.cs -> BDD: INSERT INTO Lecon (ModeleLecon, DateHLecon, EleveId, MoniteurId, Duree)
BDD --> LeconRepository.cs: success
LeconRepository.cs --> LeconMetier.cs: void
LeconMetier.cs --> LeconController.cs: return ReservationResult.Ok()
LeconController.cs --> leconService.js: 201 Created
leconService.js --> Lecons.jsx: succès
Lecons.jsx --> Secrétaire: Afficher "Leçon réservée avec succès !"
Secrétaire --> Eleve: Confirmer le rendez-vous
end
